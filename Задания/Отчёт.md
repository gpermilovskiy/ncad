### 6. Таблицы отчётов
  
  
Выполнено. Приложен файл dwg. 

  Процесс:
  
  1. Открыть файл Задание 5.dwg
  2. Выбрать инструмент "Таблица nanoCAD" во вкладке "Оформление". 
  3. В появившемся окне выбираем:
    3.1. Нестандартная
	3.2. Колонки 25 х 3
	3.3. Данные 8 х 1
  4. Нажимаем "Ок" 
  5. Размещаем таблицу справа от объектов
  6. Нажимаем на таблицу 1 раз
  7. Выбираем масштаб 1:10
  8. Нажимаем "Вставить раздел отчёта" во вкладке "Разделы"
  9. Нажимаем кнопку "Выбрать исходный объект" у появившегося раздела с отчётом. 
  10. В открывшемся окне выбираем тип объекта "Вхождение блока"
  11. Нажимаем "ОК"
  12. Заполняем отчёт
	12.1 Первая колонка шаблона отчёта = "=row-1"
	12.2 Вторая колонка = "=Object.Name"
	13.3 Третья колонка = "=Object.%название нашего состояния видимости%" (в нашем случае "Object.Видимость0")
  13. Заполняем раздел "данные" названиями колонок, согласно примеру из задания: № П/П, Имя блока, Состояние видимости
  14. Сохраняем и закрываем окно

  ### 7. Глобальные настройки универсальной выноски. 
  
Выполнено. Приложен файл dwg. 

  Процесс:
  1. Открыть "Настройки объектов" в разделе "Настройки"
  2. Открыть в появившемся окне вкладку "Символы"
  3. Открыть раздел "Выноски"
  4. Открыть подраздел "Универсальные выноски"
  5. Поменять цвет на "Красный"
  6. Открыть подраздел "Текст"
  7. Поменять текстовый стиль на "Standart" (у него включен штрифт Arial)
  8. Нажать "ОК"
  9. Выбрать инструмент "Универсальная выноска" на вкладке "Оформление"
  10. В появившемся окне заполнить:
    10.1. Первая пустая строчка: "Универсальная выноска" (новая строчка создаётся через shift+enter)
	10.2. Вторая пустая строчка: "Шрифт Arial"
	10.3. Третья пустая строчка: "Цвет красный"
  11. Нажать "ОК"
  12. Выбрать место для размещения выноски
  
  
   ### 8. Импорт и создание типов линий. 
  
Выполнено. Приложен файл dwg. 

Замечание: в импортируемом файле нет отступов между текстом, текст будет плотной линией выстраиваться. В выполненой задаче файл исправлен:
